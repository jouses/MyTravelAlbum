# MyTravelAlbum
Travel album where you can save the city name and pictures.
## Used
* Binding
* Recyclerview
* Fragment
* SqlLite
* Bitmap
* permission READ_EXTERNAL_STORAGE and READ_MEDIA_IMAGES
